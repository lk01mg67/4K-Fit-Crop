# Changelog

모든 주요 변경 사항은 이 파일에 기록됩니다.

## [1.2.0] - 2025-12-31
- **Feat**: `tqdm` 라이브러리를 도입하여 대량 이미지 처리 시 진행 상태를 시각적으로 표시 (자동 설치 지원)
- **Feat**: 파일명 생성 로직 개선 (숫자로만 된 파일명은 상위 폴더 구조를 접두사로 자동 추가, 그 외에는 원본 파일명 유지)
- **Docs**: `TODO.md` 진행 상황 업데이트

## [1.1.1] - 2025-12-31
- **Docs**: README에 CLI 고급 사용법, WebP 지원, 병렬 처리 기능 상세 설명 추가
- **Docs**: ImageMagick 설치 및 감지 로직 설명 현행화 (Fedora 포함)
- **Chore**: 불필요한 `requirements.txt` 삭제 및 README 설치 안내 간소화 (표준 라이브러리만 사용)
- **Feat**: 병렬 처리 기능 구현, CLI 인자 추가, WebP 지원 및 PEP8 리팩토링 (`6ed336d`)
- **Docs**: 문서 스타일 수정 및 TODO 업데이트 (`7971247`)
- **Misc**: ImgBot을 통한 이미지 최적화 (`104d04c`)

## [1.1.0] - 2025-12-31
- **Feat**: GitHub Actions에 Artifact 업로드 기능 추가 (`f390b26`)
- **Fix**: GitHub Actions에서 ImageMagick v6 호환성 문제 해결 (`267c317`)
- **Docs**: TODO.md 생성 및 README 라이선스 섹션 개선
- **Style**: 이미지 비율을 3:4(세로) 및 4:3(가로)로 고정

## [1.0.0] - 2025-12-30

- **Docs**: MIT 라이선스 추가 및 README 작성 (Vibe Coding 기여 명시)
- **Docs**: 재귀적 탐색 설명 복구 및 파일명 명명 규칙 변경
